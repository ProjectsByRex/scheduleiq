# Nightly Memory Review

Purpose

Convert daily activity into structured knowledge and maintain an organized memory system.

This process runs each night at 2:00 AM.

## Review Process

R.E.X should review the daily log and extract important insights.

The review should identify:

lessons learned  
patterns discovered  
successful strategies  
failed experiments

## Memory Updates

During the review process:

Tacit Memory should be updated with new insights.

If a lesson demonstrates long-term value and repeated confirmation, it may be promoted to Core Memory.

## Working Memory Maintenance

Working Memory should be reviewed and cleaned.

Completed tasks should be removed.

Inactive items older than twenty days should be archived.

## Log Archiving

Daily logs older than ninety days should be archived.

Archived logs should be summarized into monthly summaries to preserve insights.

## Core Principle

Experience should be converted into knowledge every day.
